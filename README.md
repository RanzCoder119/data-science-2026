## 📖 Tentang Repository Ini

Repository ini merupakan dokumentasi perjalanan belajar saya dalam mata kuliah **Pengantar Data Science** di Universitas Siber Asia. Saya, Ranu Ratmaja, membangun repository ini sebagai portofolio teknis yang mencerminkan proses belajar secara sistematis — mulai dari pengenalan ekosistem Python untuk data science, eksplorasi dan visualisasi data, hingga membangun model machine learning pertama menggunakan Regresi Linear.

Tujuan belajar saya dalam mata kuliah ini adalah menguasai pipeline data science secara end-to-end: dari data mentah hingga insight dan prediksi yang dapat ditindaklanjuti. Setiap notebook dirancang untuk dapat dijalankan langsung di **Google Colab** tanpa konfigurasi tambahan, sehingga siapapun dapat mereproduksi hasil yang sama.

Repository ini berisi **7 notebook Jupyter** yang mencakup topik: pengenalan Python untuk data science, exploratory data analysis (EDA), visualisasi data, algoritma klasifikasi (KNN & Decision Tree), clustering (K-Means), preprocessing data, dan regresi linear sebagai pengantar machine learning.

### 📌 Pertemuan 4 — Klasifikasi KNN

## 🎯 Kesimpulan Perjalanan Belajar Data Science Pertemuan 1–7

Tujuh pertemuan mata kuliah Data Science ini membawa saya dari nol hingga mampu membangun pipeline machine learning secara mandiri. Perjalanan dimulai dari memahami ekosistem Python (NumPy, Pandas, Matplotlib), lalu berkembang ke EDA dan visualisasi yang membantu saya "berbicara dengan data" secara visual.

Pertemuan 4 dan 5 membuka mata saya terhadap supervised learning — bagaimana mesin dapat "belajar" dari data berlabel untuk membuat prediksi. KNN mengajarkan prinsip similarity-based learning, sementara Decision Tree menunjukkan bahwa model yang baik tidak harus kompleks — cukup interpretable dan akurat. Pertemuan 6 melengkapi pemahaman dengan unsupervised learning melalui K-Means, membuktikan bahwa mesin dapat menemukan pola tanpa label apapun.

Puncaknya di Pertemuan 7 — saya berhasil membangun pipeline Regresi Linear end-to-end: dari generate data, preprocessing, training, evaluasi dengan MAE/RMSE/R², hingga interpretasi residual plot. Model mencapai R² > 0.90, membuktikan bahwa fitur yang tepat dan preprocessing yang benar adalah kunci performa model yang baik.

**Pembelajaran terpenting:** Data science bukan sekadar tentang algoritma — proses EDA, visualisasi, dan preprocessing yang matang justru menentukan 80% keberhasilan model. Saya berencana melanjutkan eksplorasi ke algoritma ensemble (Random Forest, Gradient Boosting) dan teknik feature engineering yang lebih advanced.
