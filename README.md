## 📝 Deskripsi Setiap Pertemuan

### Pertemuan 1 - Pengenalan Data Science & Python
- Pengenalan ekosistem Data Science
- Instalasi library: NumPy, Pandas, Matplotlib, Scikit-learn
- Operasi dasar array dan DataFrame
- Dataset: Iris

### Pertemuan 2 - Eksplorasi Data (EDA)
- Statistik deskriptif (mean, median, std, dll.)
- Deteksi missing values dan outlier
- Analisis korelasi antar fitur
- Dataset: Iris

### Pertemuan 3 - Visualisasi Data
- Histogram, scatter plot, box plot
- Heatmap korelasi dengan Seaborn
- Pair plot untuk analisis multivariat
- Dataset: Iris

### Pertemuan 4 - Klasifikasi dengan KNN
- Algoritma K-Nearest Neighbors
- Train-test split dan evaluasi model
- Confusion matrix dan classification report
- Dataset: Breast Cancer Wisconsin

### Pertemuan 5 - Visualisasi Data Lanjutan
- Visualisasi distribusi data
- Analisis outlier dengan IQR dan boxplot
- Visualisasi hasil clustering awal
- Dataset: Iris

### Pertemuan 6 - Persiapan Data (Data Preprocessing)
- Normalisasi Min-Max dan Standarisasi Z-Score
- Label Encoding dan One-Hot Encoding
- Feature Selection dengan SelectKBest
- Train-Test Split dengan stratify
- Dataset: Iris
